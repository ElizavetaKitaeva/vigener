The Vigenere Cipher

Лицензия программы для шифрования текста

Разработчик: Зверев С.В., Худжадзе А.Э.

Обновлено: 17.09.2020

Архитектура: x64

Интерфейс: русский

Категория: программа шифрования

Лицензия
Тип продукта: чистая установка
Количество пользователей: 1шт
Срок действия: 36 мес

Описание:
Программа предназначена для шифрования текста по Шифру Виженера. Шифр Виженера - это метод шифрования алфавитного текста с помощью серии переплетенных шифров Цезаря, основанных на буквах ключевого слова. Он использует форму полиалфавитного замещения.

1.3 Функциональное описание программного средства
1.3.1 Назначение и область применения
Данное программное обеспечение должно применятся исключительно в целях шифрования текста
1.3.2 Условия эксплуатации
1.3.2.1 Требования к техническим средствам
Данное программное средство разработано для ОС Windows и при работе занимает 7.5 МБ памяти
1.4 Ввод в действие программного средства
1.4.1 ввод в действие
Для шифрования сообщения необходимо иметь текстовые файлы для ключа и самого сообщения, также есть возможность написать текст вручную
1.5 Использование программного средства
1.5.1 Загрузка программного средства
Загрузка программы осуществляется путем скачивания инсталлятора с официального сайта программы
1.5.2 Управляющие инструкции, команды или языковые правила
1.5.2.1 управляющие инструкции и команды
После запуска программы необходимо заполнить текстовые поля для ключа и сообщение посредством открытия текстового файла или записать текст вручную
1.5.2.2 Выходные сообщения
Выходное сообщения «Файл не выбран.», показывается в том случае если была нажата кнопка «Открыть» и в диалоговом окне файл не был выбран
1.5.3 Данные
1.5.3.1 входные данные
Текст в поле сообщения и ключа или текст из файла
1.5.3.2 выходные данные
Зашифрованный текст в поле «Преобразованное сообщение»
1.5.3.3 Ограничения на данные
В данной версии программы для шифрования могут использоваться только не прописные символы из кириллицы.
1.5.4 Ошибки и сбои
Производитель не гарантирует правильность шифрования при использовании в качестве ключа или сообщения символы помимо тех что описаны в пункте 1.5.3.3.

1.7 Словарь
Сообщение – текст который необходимо зашифровать.
Ключ – Слово которое используется для шифрования.
